# 电子流萤

一个功能完整的电子木鱼网页小游戏，支持自定义标题、计数文字、图片和音频资源。

## 🎮 功能特性

### 核心功能
- **点击交互**：点击图片播放指定音频，支持触摸设备
- **计数系统**：实时显示点击次数，数据自动保存到本地存储
- **视觉反馈**：点击时显示浮动文字动画效果
- **缩放动画**：点击图片时有缩放反馈

### 自定义设置
- **标题自定义**：可修改页面顶部标题（默认：电子流萤）
- **计数文字自定义**：支持自定义计数前后文字（默认：流萤已经cia了X次llo！）
- **图片上传**：支持点击和拖拽上传自定义图片
- **音频上传**：支持点击和拖拽上传自定义音频
- **资源管理**：可选择、删除自定义资源（默认资源不可删除）

### 技术特性
- 纯HTML/CSS/原生JavaScript开发，无外部框架依赖
- 响应式设计，适配移动端和桌面端
- 音频预加载技术，实现无延迟播放
- localStorage持久化，刷新页面数据不丢失

## 🚀 快速开始

### 运行方式
1. 确保 `media` 目录下存在默认图片和音频文件
2. 使用浏览器直接打开 `index.html` 文件即可运行
3. 无需任何服务器环境，离线可用
4. 或通过链接 https://xiaocheng096.github.io/firefly-click/ 在线访问

### 环境要求
- 主流现代浏览器：Chrome、Firefox、Safari、Edge
- 支持HTML5、CSS3、ES6+

## 🎯 使用说明

### 基本操作
1. 点击中间的图片即可增加计数并播放音效
2. 点击右上角设置按钮（⚙️）打开设置面板
3. 在设置面板中可自定义标题、计数文字、图片和音频

### 计数格式
计数显示格式为：`前缀 + 计数 + 后缀`

示例：
- 默认：`流萤已经cia了0次llo！`
- 自定义前缀为"功德："，后缀为空：`功德：0`

### 文件上传
- 支持拖拽上传和点击选择文件
- 图片支持：JPG、PNG、GIF等常见格式
- 音频支持：MP3、WAV、OGG等常见格式
- 单文件大小限制：5MB

## 🔧 重置数据

在设置面板中点击「重置所有数据」按钮，可恢复所有默认设置并清空自定义资源。
